# 똑비서 (DdokBiseo)

> 모바일 우선 개인용 할일 관리 웹앱

---

## 소개

똑비서는 한 손으로 빠르게 다룰 수 있는 모바일 중심의 할일 관리 앱입니다.
모든 데이터는 사용자 기기의 브라우저에 로컬 저장되어, 외부 서버로 전송되지
않습니다. 로그인이나 회원가입 없이 바로 사용할 수 있으며, 인터넷 연결 없이도
완전히 동작합니다.

복잡한 기능을 더하기보다, 매일 쓰는 사람이 손에 익을 만한 핵심 기능만
다듬어서 만들었습니다.

---

## 주요 기능

- **달력 기반 할일 관리** — 월간 달력에서 날짜별 할일 한눈에 확인
- **주간 보기** — 한 주 단위로 중요 일정 일람
- **반복 일정** — 매일 / 매주 / 매월 / 매년 / 커스텀 요일 반복
  - 반복 일정 수정·삭제 시 "이 날짜만 / 이후 모두 / 전체" 선택 가능
- **체크리스트** — 하나의 할일 안에 여러 하위 항목 관리
- **중요도 4단계** — 색상으로 한눈에 구분
- **스와이프 제스처** — 오른쪽으로 완료, 왼쪽으로 액션 메뉴
- **드래그 정렬** — 핸들로 순서 변경
- **검색** — 제목·메모 전문 검색
- **다크/라이트/컬러 테마** — 7가지 테마 지원
- **파일 내보내기/가져오기** — 전체 데이터를 JSON으로 백업·복원

---

## 기술 스택

- Vanilla JavaScript (ES6+, 빌드 도구 없음)
- HTML5 / CSS3
- IndexedDB (모든 데이터 로컬 저장)
- Open-Meteo API (날씨 표시 — 선택적)

---

## 사용 방법

### 직접 호스팅하여 사용

1. 이 저장소를 다운로드 또는 fork
2. `index.html`이 포함된 폴더를 웹 서버에 올리거나, GitHub Pages로 배포
3. 모바일 브라우저로 접속 → "홈 화면에 추가"로 앱처럼 사용

### 로컬에서 실행

```bash
# 정적 파일이므로 별도 빌드 불필요
python3 -m http.server 8000
# 또는
npx serve
```

브라우저에서 `http://localhost:8000` 접속

---

## 데이터 저장

- 모든 데이터는 사용자 브라우저의 **IndexedDB** (`ddog-local`)에 저장됩니다.
- 테마 등 환경 설정은 **localStorage**에 저장됩니다.
- **외부 서버로 어떠한 데이터도 전송되지 않습니다** (날씨 API 제외, 위치 정보 미전송).
- 브라우저 데이터를 지우거나 다른 기기로 옮기려면 **설정 → 파일 내보내기**로
  JSON 백업을 만든 후, 새 기기에서 **파일 가져오기**로 복원하세요.
